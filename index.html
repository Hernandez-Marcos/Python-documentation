Introduction

Python is a high-level programming language, considered one of the easiest to learn and understand. It has become very popular in recent years due to its versatility and ability to solve complex problems in various areas of programming. It is widely used in web development, artificial intelligence, data science, automation, and script creation.

In this technical documentation, we will delve into the basic concepts of Python, from syntax to the most common data structures, to help readers gain a solid understanding of the language and prepare them to tackle more advanced projects. In each chapter, practical examples will be explained and exercises will be included to put into practice what has been learned. This documentation is designed for people who are just starting to learn programming with Python or who want to refresh their knowledge.

Hello World

The print() function is one of the most basic functions in Python and its use is very simple. It serves to display a message or value on the console or screen. The message or value to be displayed is written within parentheses and quotation marks, or just between parentheses if it's a variable or a value of another type, but we'll cover that later.

For example, to generate a "Hello World" with the print() function, the following would be written:

print("Hello World")

The output on the screen would be:

Hello World

It is also possible to print multiple values or messages on a single line, separating them with a comma. For example:

print("Hello", "World")

The output on the screen would be the same:

Hello World

The print() function also accepts several optional parameters such as:

  • sep: which allows to specify a separator between elements, the default value is ' ' (space).
  • end: which allows to specify the end of the print statement, the default value is '\n' (new line)

For example:

print("Hello", "World", sep=", ", end="!\n")

The output on the screen would be:

Hello, World!
Comments

What are comments in Python?

Comments in Python are lines of text used to explain code or leave notes for yourself or other developers. Comments are not part of the actual code and do not affect the operation of the program. However, they are essential to understand the code and to make it easier to maintain and modify.

Comment syntax

To write a comment in Python, use the "#" symbol at the beginning of the line. Anything you type after the "#" will be treated as a comment and will not be executed by Python. For example:

#I am a comment. And I'm here to explain that the following line of code prints "Hello World" to the console.
print("Hello World")

Practical use of comments

  • Documentation: Comments are a great way to document your code and explain its purpose and operation. This can be especially useful when other developers or even your future self need to understand the code.
  • Debugging: Comments can be used to temporarily disable a section of code, or to leave notes about a specific problem or bug.
  • Easier to read: Comments can make code easier to understand and follow, especially when it comes to complex or feature-rich code.
  • Communication: Comments are a way to communicate with other developers or team members, either to discuss a problem or to ask for help.
  • Improve maintainability: Comments help understand the code and make it easier to maintain, as they can help identify where bugs are found or understand how a specific part of the code works.

Tips for writting comments

  • Use clear and precise language.
  • Avoid unnecessary or duplicate comments.
  • Keep comments updated as the code changes.
  • Use comments to explain why you did something, not just how you did it.
  • Try to make your comments self-explanatory, that is, the code and the comments complement each other.
Variables

What are variables in Python?

In programming, a variable is a space in the computer's memory where a value is stored, which can be used and modified at different times during the execution of the program. In Python, variables are used to store and manipulate data.

Declaration and assignment of values ​​to variables

To declare a variable in Python, the "=" (equality) operator is used. The left side of the operator is the name of the variable, which can contain any combination of letters, numbers, and underscores, but must start with a letter or underscore (not a number), and the right side is the value that is assigned to the variable. For example:

#The following line of code assigns "Hello World" to a variable called hello.
hello = "Hello World"

Now we can print the variable, and as you'll see, the result is the same as printing the value itself.

print("Hello World")
print(hello)
print(hello, "Hello World", sep=" equals ")

The output on the screen would be:

Hello World
Hello World
Hello World equals Hello World

We can also change the value of the variable:

#Declaring variable and printing it
hello = "Hello World"
print(hello)
#Changing the value of the variable and printing it to show the change
hello = "Hello Universe"
print(hello)

The output on the screen would be:

Hello World
Hello Universe

Tips for Using Variables

  • Use descriptive names for variables to make it easy to understand their content and usage. For example, instead of using names like "x" or "temp", use names like "number_of_customers" or "average_temperature".
  • Avoid using Python reserved words as variable names. These words have a special meaning in the language and cannot be used as variable names.
  • Be sure to assign a value to the variable before using it. If you try to use a variable before assigning it a value, you will get an error.
  • Use variables with different names for different values, thus avoiding accidental overwriting of values.
